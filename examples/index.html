Static showcase pages built with exported JSON scenes

Examples that range from artistic wave heroes to colorful motion-led product pages.

Each page here is static HTML using the same ascii-arter package. Design in the editor, export JSON, then mount it behind an Apple-style Your Company hero, a colorful wave launch page, floating product cards, a dashboard promo, or a fintech landing page.
