Why this one exists

More randomized motion, less “big masked word in the middle”.

This demo answers the exact need for richer showcase variety: colorful background waves, layered motion, floating product surfaces and no dependency on subject text in the scene itself.

01

Color carries the mood

Instead of a monochrome art direction, Chromawave leans into candy, aurora and ocean palettes so the first screen feels lively before a screenshot even shows up.

02

No text-only canvas trick

The exported JSON uses ambient motion only. The product story lives in cards, metrics and layout — exactly how a real landing page would use the package.

  • Hero JSON subject type = none
  • CTA JSON subject type = none
  • Foreground remains pure HTML/CSS
03

Easy to remix

Open the editor, swap palettes and layer modes, export again, then keep the same page shell. It’s a useful starter for consumer brands and colorful SaaS launches alike.

Final CTA

Build the page first. Let the ASCII motion carry the atmosphere.

Chromawave is a good pattern when you want something more visual than a dashboard hero but still usable in a real product landing. The scene stays alive while the layout stays simple.
