Why this one exists

Another answer to “give me more examples, not just text.”

OrbitDeck uses ambient orbital motion and floating interface blocks so the page feels more like a product launch composition than a background with a masked label dropped on it.

01

Motion behaves like ribbon lighting

The ASCII scene is tuned to read like curved ribbons and orbit traces, giving the page depth without taking over the layout.

02

Cards tell the product story

Instead of relying on a text-mask subject, the foreground uses real UI blocks, stats and chips. That makes the demo closer to a realistic launch page.

  • Hero JSON subject type = none
  • CTA JSON subject type = none
  • Foreground still fully static-hosting friendly
03

Easy pattern to remix

Swap the copy, change the card content, keep the mount zones and export fresh JSON scenes whenever you want different motion language.

Final CTA

Use the ASCII layer like a material behind real product blocks.

OrbitDeck is for the moments when you want motion-rich pages without turning the canvas itself into the headline. Build cards, panels and pricing first — then let the orbital scene make them feel more alive.
